# Getting started with `vue-test-utils`

## Setup

``` bash
git clone https://github.com/hchiam/vue-test-utils-getting-started.git
cd vue-test-utils-getting-started
npm install
npm test
```
Note to self: Jest uses `npm` command, Tape uses `node` command.
